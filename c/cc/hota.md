## 0. 为什么 A 股必须"板块优先、个股其次"

美股选股可以"自下而上"（只看干净的基本面 + 完美的图形）。A 股不行。三个结构性原因：

1. **板块效应远强于美股**：A 股个股 60-70% 的涨跌由其所属板块驱动（行业 beta 占主导）。
   不管单只图形多好，如果板块在退潮或已被资金抛弃，它照样阴跌。
2. **政策与题材驱动**：几乎每只 A 股龙头的启动都伴随政策催化或题材爆发；
   一只"逆主线"的股票几乎从不能独立持续上涨。
3. **资金集中且情绪化**：北向 + 公募 + 游资同时聚焦于少数几条主线；
   其余一切都陷入流动性枯竭。

**结论**：先识别当前的主线板块（<= 3 条），再在其中挑龙头 / 二线，
最后套用 framed.md 的过滤器（§2 选股 + §3 入场触发）。如果板块选错，再精细的个股筛选也救不了。

## 1. 热门板块的 5 阶段生命周期

借用 Weinstein 的 4 阶段，但板块在其之上多了一个"孕育"阶段
（因为资金共识需要时间形成）：

| 阶段 | 特征 | 资金动向 | 动作 |
|-------|------|----------|------|
| **孕育** | 1-2 只票意外涨停；板块指数尚未启动；无媒体报道 | 游资试探；无机构 | 观察；不参与 |
| **启动** | 龙头 2-3 连板；板块指数放量突破 60 日均线；二线跟随 | 北向 / 公募开始流入 | **最佳入场窗口** |
| **主升** | 板块指数斜率陡峭；普涨；新龙头每日接力 | 持续流入；融资余额上升 | 已有仓位兑现收益；新仓位需更紧止损 |
| **高潮** | 媒体头条 / 论坛热度爆表 / 散户搜索量飙升 / 卖方分析师集体上调目标价 | 散户净流入；机构开始减仓 | 分批出货；不做接力 |
| **退潮** | 龙头炸板 / 一字跌停 / 最高连板数崩塌 / 北向流出 | 资金流出；融资平仓 | 清掉剩余 |

记住：**只在"启动"阶段入场；"主升"阶段持有但不开新仓；"高潮 / 退潮"阶段严格出场**。

## 2. 五维识别（资金 / 价格 / 题材 / 情绪 / 供应链）

一个真正的热门板块至少同时在**3 个维度**上得到确认。单维度信号 = 假热度 / 一日游。

### 维度 A：资金流（最难的信号）

按可信度从高到低：

1. **北向行业净流入排名**：连续 3-5 天某行业单日净流入 >= 5 亿元 = 强信号。
   北向是 A 股里最聪明的钱 —— 机构 + 偏多。跟随它很少出大错。
2. **公募持仓变动**（季报 / 半年报披露后）：某板块连续 2 个季度仓位增加 >= 2 个百分点
   = 趋势信号。滞后但权威。
3. **龙虎榜机构席位**：某只票连续 2-3 个交易日有 >= 2 个机构席位净买入（每个 >= 3000 万）
   = 机构进场信号。叠加板块普涨 = 板块级机构共识。
4. **主力资金净流入**（同花顺 / 东方财富口径）：板块连续 3 天净流入 >= 20 亿元
   = 资金共识。注意：此指标含游资 —— 可信度低于北向 —— 需与其他信号配合。
5. **融资余额变动**：板块融资余额连续 5 个交易日上升 >= 5% = 杠杆资金涌入。
   双刃剑：早期有益；高位时是退潮的前兆。
6. **ETF 申购 / 赎回**：板块 ETF 连续多日净申购 = 增量资金；持续溢价 >= 1%
   = 散户在涌入（顶部信号）。

### 维度 B：价格行为

1. **板块指数突破**：板块指数（申万一 / 二级、同花顺概念、东方财富行业）突破
   前 6 个月高点 + 周线收阳 + 放量 = 启动信号。
2. **涨停集群**：同板块单日 >= 5 只涨停 = 板块异动；>= 10 只 = 主流方向。
   必须连续 2-3 天才算真启动。
3. **连板高度**：龙头 >= 3 连板 = 预期会发酵；龙头炸板 = 退潮前兆。
4. **板块 RS（相对强度）**：板块指数 30 日涨幅排在申万一级前 5 = 资金聚焦于此。
5. **成交占比**：板块单日成交额连续 3 个交易日 >= 全市场成交的 10% = 主线确认。
6. **二三线响应**：龙头涨停后，板块内二线次日跟随 >= 5% 涨幅 = 板块效应成立。
   若只有 1-2 只动 = 个股行为，非板块行为。

### 维度 C：题材催化（叙事）

一个真正的热门板块必须有清晰的"为什么是现在"的叙事。无叙事 ≈ 纯炒作 ≈ 一日游。

催化类型：

1. **政策催化**：国务院 / 发改委 / 工信部 / 财政部重磅文件、行业规划、补贴计划。
   政策日历：3 月两会、7 月政治局会议、10 月四中全会、12 月中央经济工作会议（CEWC）。
2. **行业事件**：龙头技术突破、量产里程碑、大额海外订单、专利公告。
3. **海外映射**：
   - NVIDIA / OpenAI 重大发布 -> 国产算力、AI 应用
   - 特斯拉 Optimus 进展 -> A 股人形机器人
   - SpaceX 发射 -> 商业航天
   - 美光 / 三星涨价 -> 国产存储
   - 西方制裁升级 -> 自主可控（半导体、操作系统）
4. **业绩驱动**：龙头业绩预告超预期 -> 板块跟随（尤其供应链上下游）。
5. **周期驱动**：大宗商品突破（铜、铝、稀土、油气）-> 上游资源股。
6. **重大会议预期**：会前抢跑（交易预期），会后兑现或证伪（交易落地 / 杀预期）。

### 维度 D：情绪与热度

主要用于判断"位置"，而非"方向"。

正面（启动）信号：
- 雪球 / 东方财富股吧讨论上升但未到顶
- 微信指数 / 百度指数温和上行
- 卖方开始密集发布该板块的深度报告

反向（顶部）信号 —— 集齐就减仓：
- 财经媒体头版（财联社、新华财经、证券时报）连续多日看多
- 论坛热榜上有 >= 5 只该板块的票
- 出租车司机 / 同事 / 朋友主动谈论它
- 微信指数飙升 >= 5 倍
- 公募集中发行主题基金（经典顶部信号）

### 维度 E：供应链验证（最慢但最实在）

资金会说谎，价格会说谎 —— 供应链数据不会（且最难造假）。

- **上游价量**：原材料、关键元器件涨价或短缺（如 HBM、光模块芯片、稀土）。
- **中游订单**：行业协会月度数据、海关进出口、上市公司月度经营报告、招标公告。
- **下游验证**：终端市场销量数据（新能源车、AI 服务器出货）、海外厂商指引。
- **基本面拐点**：全行业 ROE / 毛利率向上拐头。

**供应链验证滞后于资金流，但一旦确认 = 主升浪可持续 6-18 个月。**
纯资金驱动、无供应链确认的上涨最多持续 1-3 个月。

## 4. 扫描 SOP（固定的每日 / 每周流程）

### 4.1 盘前扫描（每日 8:30-9:15，15 分钟）

1. **隔夜海外**：纳斯达克、NVIDIA、特斯拉、美光 + 大宗（铜、油、稀土）。
   映射可能受影响的 A 股板块。
2. **隔夜政策 / 重大事件**：财联社快讯、新华财经头条。识别任何黑天鹅或新主线催化。
3. **晨会卖方**：快速扫一遍晨报（不依赖；只检查有无一致方向）。
4. **个人自选预警**：检查自选股盘前公告（业绩预告、内部人减持、监管行动）。

### 4.2 盘中扫描（10:00、11:00、14:00、14:50 —— 各 5 分钟）

不盯实时盘口。只在这四个节点瞄一眼：

1. **10:00**：开盘 30 分钟后，板块排名前 5 是否稳定（避免被开盘脉冲骗到）？
2. **11:00**：上午收盘前 —— 涨停家数、最高连板高度、板块普涨广度。
3. **14:00**：午后开盘后，有无"下午发酵"的新板块（A 股常见的尾盘异动）？
4. **14:50**：收盘前 10 分钟 —— 封板状态、龙头是否炸板、机构席位是否在抢筹？

### 4.3 盘后复盘（每日 16:00-17:00，30-60 分钟）

固定模板（按顺序）：

1. **三大指数**：沪深300、创业板、上证综指收盘 -> 更新 framed.md §1 指数与情绪闸门。
2. **板块涨幅排名**（同花顺概念 + 申万一级）：记录涨幅前 5 + 每板块涨停家数。
3. **涨停板分析**：按板块汇总 —— 识别"主流方向"、"补涨方向"、"分歧方向"。
4. **资金流**：
   - 北向单日净流入 + 行业前 5 + 净买额个股前 10
   - 主力资金板块净流入前 5
   - 龙虎榜机构席位扫描（重点：连板龙头、放量突破票）
5. **跟踪表更新**：在板块跟踪表加上今天这一行；剔除已退潮板块（见 §6 模板）。
6. **明日计划**：识别 1-2 个可能继续发酵的方向；列出其龙头及入场 / 止损价。

### 4.4 周末深度复盘（周六，2-3 小时）

1. **周度板块 RS 排名**：申万一级板块 13 周 / 26 周涨幅排名变化。
2. **下周政策日历**：即将到来的重要会议 / 文件预期。
3. **业绩窗口检查**：下周哪些重点公司发布业绩。
4. **海外映射回顾**：本周关键的 NVIDIA / 特斯拉 / OpenAI 时刻。
5. **持仓复盘**：对照 framed.md §4 出场规则检查每只持仓；结构仍完好吗（站上 5 日均线、未触发时间止损 / 价格止损）？需要减仓吗？
6. **自选更新**：更新 watchlistd.md 池（最多 6 只）。

## 5. 真假热门板块：4 项印证

满足 >= 3 项 = 真主线；<= 2 项 = 假热度 / 题材炒作，不参与。

| 维度 | 真热 | 假热 |
|------|------|------|
| 持续性 | 资金持续流入 >= 5 个交易日 | 1-2 天脉冲，第 3 天破位 |
| 广度 | 板块内 >= 10 只股响应（>= 5% 涨幅） | 只有龙头 1-2 只动 |
| 资金性质 | {北向、公募、机构龙虎榜席位} 中至少 2 项确认 | 纯游资席位，无北向、无机构 |
| 叙事质量 | 清晰的政策 / 行业 / 海外催化链条 | "题材炒作"、"蹭热度"，无实质 |

**两个额外排除信号**（任一出现 -> 放弃）：

- 板块龙头出现"天地板"（开盘 +10% 涨停，收盘 -10% 跌停）或"冲高跳水"
  （高开 +10%，收盘 -10%）。
- 头版媒体连续 3 天报道 + 公募发行新主题基金 = 已进入高潮阶段；**不要入场**。

## 7. 龙头识别：从板块到个股

确认一个板块为真热后，按以下顺序锁定龙头：

### 7.1 龙头排序优先级

1. **首板 / 首个突破前高**（情绪龙头）
2. **业绩与题材双纯**（基本面龙头）：业绩超预期 + 主业 100% 映射此题材
3. **自由流通市值在 50 亿-500 亿元区间**（资金友好）：太小 = 易被操纵，太大 = 推不动
4. **龙虎榜机构 + 北向同时出现**（资金一致）
5. **连板数最高 + 封板最早 + 换手适中（3-15%）**

### 7.2 "龙头 vs 跟风"快速过滤

| 特征 | 龙头 | 跟风 |
|------|------|------|
| 涨停时间 | 上午 10:00 前 | 下午，甚至尾盘 |
| 封板厚度 | 厚（封单 / 流通盘 >= 1%） | 薄，反复开板 |
| 业绩 | 已验证 / 强预期 | 蹭概念，主业无关 |
| 历史地位 | 之前已是子板块龙头 | "突然冒出来"的票 |

**不买跟风票**：它们的炸板率明显高于龙头，且即便板块延续，
跟风票的涨幅也大幅落后于龙头。

### 7.3 让候选股过 framed.md

锁定 1-2 只候选龙头后，逐一过 framed.md：

- §1 闸门 -> 入场前指数与情绪环境必须是绿灯（非退潮 / 冻结）
- §2 硬性过滤 -> 自由流通市值 50 亿-500 亿、有量、题材龙头 / 扎实老二；排除 ST、5+ 连板、业绩前的票
- §3 入场触发 -> 必须存在明确触发：放量回踩 5/10 日均线站稳，或放量突破平台（>= 均量 1.5 倍）
- §5 仓位 -> 双槽集中模式：每槽 ≈ 25,000 元（各占账户约 50%），全部 50K 可部署。
  每槽风险由 ATR 止损宽度决定（framed.md §4A），而非 1% 上限。同时检查流动性：
  持仓 ≤ 该股 10 日平均成交额的 1%。

通过 -> 加入 watchlistd.md；在 §3 触发时入场，不提前。

## 8. A 股热门板块的 6 类常见驱动模板

每个模板有不同的入场节奏和持续时间；识别模板有助于你设定合理的止盈 / 止损周期。

### 模板 1：政策催化

- 触发：国务院 / 部委发布文件、规划、补贴计划
- 节奏：文件发布日板块爆发，主升 1-2 周，1-3 个月内分化
- 案例：低空经济、新质生产力、人形机器人扶持政策
- 持续：1-6 个月（取决于政策力度 + 供应链落地）
- 入场：文件后 1-3 个交易日，龙头放量突破平台（framed.md §3B）时
- 出场：政策预期兑现 + 媒体一致看多 + 公募发行主题基金

### 模板 2：业绩驱动

- 触发：龙头业绩预告 / 报告大幅超预期
- 节奏：公告日跳空高开，3-5 天确认主升，供应链 1 个月内跟随
- 案例：历史案例 —— 宁德时代、立讯精密、北方华创
- 持续：3-12 个月（基本面驱动；最长）
- 入场：公告后回踩站稳 + 周线放量突破
- 出场：业绩增速拐头（增速连续 2 个季度下滑）

### 模板 3：海外映射

- 触发：海外龙头（NVIDIA / 特斯拉 / OpenAI / SpaceX）重大公告
- 节奏：海外夜间发布；A 股次日早盘跳空；发酵 2-5 天；1-2 周内回踩
- 案例：NVIDIA GTC -> 算力板块；特斯拉 AI Day -> 机器人；SpaceX 发射 -> 商业航天
- 持续：1-3 个月（情绪驱动）
- 入场：不追跳空日；等 3-5 天龙头缩量回踩 + 放量重启时
- 出场：海外标的回调 + A 股龙头炸板

### 模板 4：周期驱动

- 触发：大宗商品突破（铜、铝、稀土、油气）或上游材料涨价
- 节奏：大宗领先 1-2 周；股票滞后跟随后加速
- 案例：历史走势 —— 紫金矿业、北方稀土、洛阳钼业
- 持续：3-12 个月（取决于商品周期长度）
- 入场：大宗突破 60 周均线 + 龙头周线放量
- 出场：大宗见顶 + 板块出现出货特征

### 模板 5：自主可控 / 制裁驱动

- 触发：制裁升级、技术出口管制收紧、外交摩擦
- 节奏：消息日跳空高开；发酵 2-3 天；1-2 周内分化
- 案例：华为供应链、半导体设备、操作系统、信创
- 持续：3-12 个月（结构性题材；反复激活）
- 入场：消息后回踩站稳 + 龙头创新高
- 出场：消息缓和 + 题材降温

### 模板 6：重大会议预期

- 触发：两会、中央经济工作会议、政治局会议、三中全会
- 节奏：会前 2-4 周交易预期；会议期间兑现或证伪
- 案例：CEWC 前"稳增长"；两会前"新基建"
- 持续：1-2 个月（事件驱动；最短）
- 入场：会前布局（高风险），或会后挑"最明确受益"的方向
- 出场：会议结束后 1-2 周 —— 预期兑现，出场

## 9. 入场时机：板块阶段 × framed.md §3 触发矩阵

|                                  | 孕育 | 启动         | 主升           | 高潮 | 退潮 |
|----------------------------------|------|--------------|----------------|------|------|
| **§3B 放量突破平台 / 前高**      | 否   | **最佳入场** | 可以，更紧止损 | 否   | 否   |
| **§3A 放量回踩 5/10 日均线站稳** | 否   | **次优**     | 可以           | 否   | 否   |
| **§3C 强势龙头接力（半仓）**     | 否   | 仅限暖市     | 暖市，更高风险 | 否   | 否   |
| **连板（5+）**                   | 不追 | 不追         | 不追           | 不追 | 不追 |

**核心原则**：只有当板块在"启动"（或主升早期）"且"个股触发 framed.md §3 时才入场 ——
§3B 放量突破平台 / 前高（>= 均量 1.5 倍）最干净，§3A 放量回踩 5/10 日均线站稳次之。
§3C 接力只用半仓、仅限暖市。绝不追 5+ 连板（framed.md §2C 自动排除）；"高潮 / 退潮" = 不开新仓。

## 10. 板块退潮的提前识别（领先信号）

板块退潮通常领先板块指数破位 1-2 周；提前发现可避免被套在主升末段：

1. **龙头炸板**：板块龙头从涨停跌到 -5% 或更差 = 主升结束。
2. **连板高度崩塌**：最高连板数从 5-7 板降到 2-3 板，且新连板票出现畸高换手。
3. **二线接力质量下降**：之前"龙头涨停 -> 二线 +7%"，变成"龙头涨停 -> 二线 +2%"。
4. **跌停 / 炸板家数上升**：板块单日 >= 3 只炸板 = 资金分歧明显。
5. **北向开始流出**：北向从净买转为净卖 >= 3 亿元，连续 3 个交易日。
6. **媒体狂欢**：财联社 / 央视财经头版 + 公募发行主题基金 —— 全部聚集在这个题材上。
7. **散户搜索飙升**：百度指数 / 微信指数 5 天内翻倍。
8. **题材延伸到"极端蹭概念"**：连主业完全无关的票都涨停 = 资金耗尽前的最后狂欢。

**当 6 / 7 / 8 同时出现时，几乎 100% 是顶。**

## 11. 风险红线（不参与的板块 / 形态）

1. **"天地板"龙头**：开盘涨停后收盘跌停 -> 板块情绪反转；3-5 天不入场。
2. **连续高位一字板（>= 5 板）后开板**：游资游戏；散户接力 = 接飞刀；绝对不碰。
3. **板块指数已涨 >= 50% 且启动以来 >= 3 个月**：主升已过半；性价比太薄。
4. **头版媒体 + 公募发主题基金 + 散户狂热 —— 三者齐现** = 顶；只减不加。
5. **纯炒未兑现的政策预期**：会前最后一周不开新仓（兑现风险高）。
6. **个股层面**：ST / *ST / 被调查 / 商誉减值预警 / 大股东减持 -> 板块再热也不入场。
7. **指数环境第 3 档**（派发日 >= 6、iVX > 32）-> 任何热门题材都不参与；现金为王。

## 14. 个人复盘与迭代

每月最后一周，做一次"板块识别能力"自查：

1. **命中率**：本月识别的 N 个热门板块中，有多少真的进入了主升？目标 >= 60%。
2. **漏判率**：本月真正进入主升的板块中，你漏掉了多少？目标 <= 20%。
3. **误判率**：你入场后发现是假热度的次数？目标 <= 1 次/月。
4. **出场时机**：高潮 / 退潮信号出现后，你延迟了多少天才出场？目标 <= 3 天。

任一维度未达标，下月针对性强化（如漏判率高 -> 加强盘前海外扫描；
误判率高 -> 收紧 §5 印证标准）。

## 附录 A：常见误区

1. **"板块在涨，我追"**：忽视阶段；追进高潮 / 退潮必然套牢。
2. **"龙头涨太多了；二线有价值"**：A 股二线弹性远低于龙头，且更易炸板。
3. **"板块逻辑扎实；我逢跌加仓"**：板块逻辑不变 ≠ 趋势不变；一旦趋势破，再强的逻辑也低头。
4. **"这板块上次爆发过；再来一次"**：板块的二次启动远不如新板块首次启动概率高，且要求更严的印证。
5. **"政策出了 —— 我冲进去"**：政策发布日往往是预期顶点；等回踩验证。
6. **"公募在加仓；我跟"**：公募披露滞后 1-3 个月；跟它往往是买在高潮。
