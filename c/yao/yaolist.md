## 每日操作规则

**盘前扫描 (8:30-9:00):**
1. 跑 `python c/ctool/yao_gate.py` — 妖股专用闸门 (池内涨停数 + 指数极端崩盘检测)
   - RED = 池内0涨停 或 任一指数跌 >3% → 今天不动
   - AMBER = 池内1-2涨停 → 谨慎, 最多选 1 只
   - GREEN = 池内3+涨停 → 进攻, 最多选 2 只
2. 用 Sina API 拉 25 只现价 + 昨日是否涨停
3. 筛选 2-3 只: 昨日涨停 + 首封早 (<10:00) + 换手适中 (5-20%)

**入场:**
- T+1 开盘价买入，每只 ¥25,000，向下取整 100 股
- 若开盘一字板 (>+9.5%) → 放弃该票

**出场 (先触发者执行):**
- 3 个交易日无新涨停 → 次日开盘清仓
- 收盘跌破入场价 -5% → 次日开盘清仓
- 浮盈 +10% → 次日开盘清仓（吃完就走）
- 炸板未回封 → 次日开盘清仓

**仓位:**

- 闸门 RED → 空仓
